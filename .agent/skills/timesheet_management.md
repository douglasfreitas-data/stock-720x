# Timesheet Management

## Goal
Accurately track time spent on tasks for project management and billing.

## Instructions

1.  **Locate `TIMESHEET.md`**:
    -   The file is located at the root of the project: `TIMESHEET.md`.

2.  **Entry Format**:
    -   `| Date | Time | Duration | Task Description | Status |`
    -   Example: `| 2026-02-07 | 14:00 - 16:00 | 2h | Refactoring authentication flow | Completed |`

3.  **When to Update**:
    -   At the start of a task (optional, can mark start time).
    -   **MANDATORY**: At the end of a task or session.

4.  **Calculation**:
    -   Ensure the duration is calculated correctly based on start and end times.
    -   Update the "Total Hours" at the bottom of the file if it exists.

## Triggers
- "Update timesheet"
- "Log my time"
- "How long did this take?"
