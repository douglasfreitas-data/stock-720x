## Formato

Cada ADR segue o formato:
- **Status**: Proposto, Aceito, Substituído, Obsoleto
- **Contexto**: Por que a decisão foi necessária
- **Decisão**: O que foi decidido
- **Consequências**: Impactos positivos e negativos
